# Tech Professionals Financial Behavior: Data Analysis Study

Analysis of financial satisfaction and spending patterns among technology professionals

## Overview
- 75 respondents from data and analytics roles
- Salary ranges from <100k to 500k+ RUB
- Key finding: satisfaction plateaus around 251-300k RUB

## Key Findings
- Financial comfort doesn't increase linearly with salary
- Top spending categories: Food (57%), Housing (44%), Investments (35%)
- 87% have emergency fund, 20% have 1+ year savings

## Methodology
Survey design, data collection, statistical analysis

## Tech Stack
Python, pandas, matplotlib, survey design
